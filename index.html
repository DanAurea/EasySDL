EasySDL 0.6

CMake is needed. Install included.

$ git clone https://github.com/Ousret/EasySDL/
$ cd EasySDL
$ cmake -G"Unix Makefiles"
$ make
$ sudo make install

        #include <ESDL.h>
CFLAGS = `sdl-config --libs --cflags` -lESDL
